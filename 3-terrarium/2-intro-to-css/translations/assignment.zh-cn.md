# 重构 CSS

## 简介

使用 Flexbox 或 CSS Grid 重新规划盆栽盒，拍几张在不同浏览器上运作的画面。重新规划时，你可能需要改变程式码中版本的标记。不需要去考虑植物拖曳的问题，我们只重构 HTML 与 CSS 的部分。

## 学习评量

| 作业内容 | 优良                                    | 普通             | 待改进               |
| -------- | --------------------------------------- | ---------------- | -------------------- |
|          | 使用 Flexbox 或 CSS Grid 呈现新的盆栽盒 | 重新取代部分元素 | 无法更新原有的盆栽盒 |
